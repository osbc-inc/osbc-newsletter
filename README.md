## 🏢 About OSBC

오에스비씨(OSBC)는 국내 최초·최다 오픈소스 컨설팅 및 솔루션 공급 이력을 보유한 오픈소스 전문 기업입니다.

2006년 설립 이후 다음 분야에서 기업의 실질적인 대응 체계 구축을 지원하고 있습니다:

- 오픈소스 보안
- 라이선스 컴플라이언스
- SBOM 구축
- 공급망 보안 관리
- 규제 대응 컨설팅
